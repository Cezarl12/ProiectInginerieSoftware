# SportMap (ActiveZone)

Multi-sport platform with interactive map - centralizes sports venues and
facilitates real-time match organization.

## Repository Structure

This is a monorepo containing both backend and frontend:

- **`backend/`** - ASP.NET Core 8 Web API (Clean Architecture)
- **`frontend/`** - Angular 19 SPA with Tailwind CSS

See:
- [`backend/README.md`](backend/README.md) for backend setup
- [`frontend/README.md`](frontend/README.md) for frontend setup
